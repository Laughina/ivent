# Ivent Bot

### Описание

Telegram бот для отслеживания актуальных мероприятий в городе Якутск. Бот получает данные с API appmost.ru и afisha.ykt.ru и отображает их в удобном виде с возможностью просмотра деталей и покупки билетов.
